# Stack Tecnológica
O projeto baseia-se na stack moderna do ecossistema Laravel:
- **Backend/Core**: Laravel (v12 candidate).
- **Frontend Interativo**: Livewire (v4 candidate) e Alpine.js.
- **Estilização**: Tailwind CSS com compilação dinâmica de tokens.
- **Banco de Dados**: PostgreSQL com uso intensivo de `jsonb` para esquemas de componentes e snapshots.
- **Governança de IA**: AI Dev Superpowers (Orquestrador, Agentes Especializados e Skills).
